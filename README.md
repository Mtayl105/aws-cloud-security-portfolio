# AWS Cloud Security Portfolio - AWS Hub

This repository contains a portfolio of AWS cloud security projects, demos, and reference material collected by Mtayl105.

## Overview

The "AWS Hub" is a collection of small projects demonstrating best practices, tools, and patterns for securing AWS environments. Expect example IaC, audit scripts, hardening guides, and threat modelling notes.

## Contents

- 01-infrastructure/  - Terraform and CloudFormation examples for secure baseline setups
- 02-auditing/        - Scripts and tools for auditing AWS accounts (IAM, S3, CloudTrail, Config)
- 03-hardening/       - OS and service hardening guides
- 04-demos/           - Small hands-on demos showing security features (GuardDuty, Macie, Config rules)
- docs/               - Additional documentation and runbooks

## Getting started

1. Review each project's README under its directory for setup and requirements.
2. Use a dedicated AWS account or sandbox when running demos.
3. Inspect IaC before applying; follow least-privilege and secure defaults.

# AWS Cloud Security Portfolio - AWS Hub

This repository contains a portfolio of AWS cloud security projects, demos, and reference material collected by Mtayl105, split into a week by week learning basis.

## Contents 

- 01-infrastructure/  - Terraform and CloudFormation examples for secure baseline setups
- 02-auditing/        - Scripts and tools for auditing AWS accounts (IAM, S3, CloudTrail, Config)
- 03-hardening/       - OS and service hardening guides
- 04-demos/           - Small hands-on demos showing security features (GuardDuty, Macie, Config rules)
- docs/               - Additional documentation and runbooks
